*** IMPORTANT — FIRST TIME SETUP ***
After importing database.sql, open this URL first:
http://localhost/melt-and-more/setup.php

This creates the admin account with correct password.
Delete setup.php after running it once.

HOW TO RUN (XAMPP):
--------------------
1. Install XAMPP (xampp.apachefriends.org)
2. Start Apache and MySQL
3. Put "melt-and-more" folder inside XAMPP's htdocs
4. Open phpMyAdmin: http://localhost/phpmyadmin
5. Create new database: "melt_and_more"
6. Import "database.sql"
7. Run setup: http://localhost/melt-and-more/setup.php
8. Visit: http://localhost/melt-and-more/
